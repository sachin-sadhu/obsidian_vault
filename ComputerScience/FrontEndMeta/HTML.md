# Hypertext Markup Language (HTML)

## Adding Images

<img src="link to image file" alt="image">

Can add width and height attributes to change dimensions of an image

# Tables

```html
<table>
    <tr>
        <th>Heading1</th>
    </tr>
    <tr>
        <td>Heading1</td>
    </tr>
</table>
```
