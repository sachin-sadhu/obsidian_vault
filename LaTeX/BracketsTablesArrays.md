
# Tables
    
\begin{tabular}{ c | c | c }
x & y & z \\ \hline
\end{tabular}

\\ & is used to serperate rows, parameter passed as { c | c | c} is used to denote colums and seperators.
\\ hline is used to add a horizontal line
